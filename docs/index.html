Where does the data come from?

To determine baby name popularity, I turned to data from the Social Security Administration. Yes, when a baby is born in the United States, a very special thing happens: they get a Social Security number. This means that the SSA has been keeping track of baby names since 1880.

The SSA releases this data annually, and the most recent data available is from 2024. The dataset includes the year, name, sex (recorded as F or M), and the number of babies born with that name.

What names are included?

For privacy reasons, the SSA only includes names that were given to at least 5 babies in a given year in any geographic area. This means that very rare or unique names will not appear in the dataset.

You can read more about the SSA baby names data and download it yourself at ssa.gov/oact/babynames.
